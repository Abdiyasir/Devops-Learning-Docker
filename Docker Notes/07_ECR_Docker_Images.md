# Pushing Docker Images to Amazon ECR

## What is ECR
- Amazon Elastic Container Registry (ECR) is a fully managed private Docker registry
- Used for storing and managing private Docker images inside AWS

## How To Use ECR

**1.** Create an ECR Repository

**2.** Retrieve Push Commands

**3.** Authenticate Docker to AWS ECR using commands on the website 

**4.** Build Your Docker Image

**5.** Tag Your Image for ECR

**6.** Push Image to ECR

**7.** Pull the Image From ECR

**8.** Run the Pulled Image
